SmoothInputLayout
=========

介绍
---

仿微信式，平滑输入面板，防止键盘的出现与消失导致特殊输入面板的顶起与塌陷。

使用
---

**引用:**
```
dependencies {
    ...
    implementation 'io.github.alexmofer.widget:smoothinputlayout:1.1.2'
    ...
}
```
